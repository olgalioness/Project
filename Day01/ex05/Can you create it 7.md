1. Создайте файл, который содержит только “42” и НИЧЕГО больше.
   > Create a file containing only "42", and NOTHING else.

2. Его название будет:
   > Its name will be:
```
“\?$*’KwaMe’*$?\”
```

3. Example:
```
$>ls –lRa *waM* | cat –e
```
